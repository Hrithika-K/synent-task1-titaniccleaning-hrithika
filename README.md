# Titanic Data Cleaning and Preprocessing

## Objective

The objective of this project is to clean and preprocess the Titanic dataset to make it suitable for analysis.

## Dataset

Titanic Dataset

## Tools Used

* Python
* Pandas
* Google Colab

## Tasks Performed

* Loaded the dataset
* Identified missing values
* Filled missing values in Age and Embarked columns
* Removed the Cabin column due to excessive missing data
* Removed duplicate records
* Generated a cleaned dataset

## Results

The dataset was successfully cleaned and prepared for further analysis.

## Conclusion

Data cleaning is an important step in the data science workflow. The cleaned dataset can now be used for visualization and machine learning tasks.
